# 防抖 debounce

## 原理

在事件被触发n秒后再执行回调，如果在这n秒内又被触发，则重新计时。

## 使用场景

- 按钮提交场景：防止多次点击提交按钮，只执行最后提交的一次

- 搜索框联想场景：防止联想发送请求，只发送最后一次输入

## 简易版实现

```javascript
function debounce(fn, wait) {
  let timeout;
  return function () {
    const context = this;
    const args = arguments;
    clearTimeout(timeout);
    timeout = setTimeout(function () {
      fn.apply(context, args);
    }, wait);
  }
}
```
